# FlowCash

Simple personal finance tracking application built with React and TypeScript.

---

## About the Project (EN)

FlowCash is a frontend application developed with **React**, **TypeScript**, and **Vite** that allows users to register and visualize financial transactions.

The application interacts with a **local mock API** powered by JSON Server. The goal of the project is not to simulate a production backend, but to demonstrate core frontend development patterns using a realistic data flow.

This project focuses on practical concepts such as:

* Global state management using React Context API
* HTTP requests handled with Axios
* Type-safe data modeling with TypeScript
* Component organization and separation of concerns
* Basic CRUD operations using a local JSON Server

Recruiter note: running the project locally is optional. The screenshots below illustrate the main interface and workflow.

---

## Sobre o Projeto (PT-BR)

FlowCash é uma aplicação frontend desenvolvida com **React**, **TypeScript** e **Vite** que permite registrar e visualizar transações financeiras.

A aplicação utiliza uma **API simulada localmente** com JSON Server. O objetivo do projeto não é reproduzir um backend de produção, mas demonstrar padrões fundamentais de desenvolvimento frontend utilizando um fluxo de dados realista.

O projeto explora conceitos práticos como:

* Gerenciamento de estado global com React Context API
* Requisições HTTP utilizando Axios
* Modelagem de dados com tipagem forte em TypeScript
* Organização de componentes e separação de responsabilidades
* Operações básicas de CRUD utilizando JSON Server local

## Running Locally (Optional)

```bash
git clone https://github.com/italocbrito/flowcash.git
cd flowcash
npm install
npm run dev
```

Start the local JSON Server (if configured):

```bash
npm run server
```
